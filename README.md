## CubeSolver 1.2

Ein ungelöster Zauberwürfel sieht einfach "unordentlich" aus – deswegen kann man mit meiner Webseite komplett ohne Kenntnisse den Rubik's Cube lösen! Einfach die Farben angeben, und die Lösung wird generiert und abgespielt. Durch Anklicken der verschiedenen Notationen kann innerhalb der Lösungsanimation zurück gespult werden und Stück für Stück der Cube gelöst werden.

Beachte:
Achtet beim eingeben der Farben immer auf die anderen Mitten um die aktuelle Seite - eine falsche Eingabe und der Cube kann nicht gelöst werden. Vor allem beim Übergang auf die Obere / Untere Seite!
Die Webseite funktioniert nur mit dem Western Color Scheme. Wenn also die voreingestellten Mitten nicht zu eurem Cube passen, gibt es (noch) keine Option, diesen zu lösen. Sollte aber in Europa kaum vorkommen.

Technologien:
Frontend: NextJS, Tailwind, TypeScript, ThreeJS
Backend: FastAPI, kociemba (Algorithmus zum Finden der Lösung)
